How to make box models:

  1. In CSS we have this idea of the box model which has the content, it has the padding, the border which we’ve seen before, and then the margin.

  2. Borders can be applied to most HTML elements within the body. The border property defines the space between an element’s padding and margin.

  3. The border width can be adjusted using the element Border-width.

  4. border-style: The values can be solid, dotted, dashed, double, groove, ridge, inset and outset.

  5. Border-color: defines the color of the border

  6. The border property has 4 possible sides: border-top, border-bottom, border-left, border-right If you want to set a border on only one of the four sides, you need to include the border’s position in the CSS property. For example, for a bottom border, you can write:

    7. For example,

    p {
    border-color:pink;
    border-bottom-style: double;
    border-bottom-width: 4px;
    }

Additional CSS properties:

  • Let’s quickly examine one more properties pertaining to the box model: box-sizing First, take a look at this example:
    • div {
    box-sizzling: border-box;
    }


By default, the box-sizing property is set to content-box, which means that the width and height property values correspond to the width and height of the content area only. When set to border-box, however, the width and height property values correspond to the width and height of content + border + padding.
